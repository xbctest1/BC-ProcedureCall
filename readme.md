## 白菜通信过程调用框架

### 简介

仓库包含三个主要项目及其下属的各子项目

- 易语言实现的协程本体（基于Win32Fiber）与协程扩展（文件读写、TCP客户端）
- IPC（进程间通信，基于Win32API邮槽和共享内存实现，可采用APC/协程模型或IOCP/多线程模型）
- RPC（远程过程调用，易的实现模型同上，但关键在于跨语言能力，目前实现或已部分实现的语言有E、PHP、JS、Python）
